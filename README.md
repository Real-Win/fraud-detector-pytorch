### 🏦 Projet 2 — Détecteur de Fraudes (Kaggle + SMOTE)
- **Stack** : PyTorch, Scikit-learn, imbalanced-learn
- **Modèle** : MLP (2 791 paramètres)
- **Dataset** : 284 807 transactions réelles (Kaggle)
- **Technique** : SMOTE pour équilibrage des classes

| Métrique | Sans SMOTE | Avec SMOTE |
|----------|-----------|------------|
| Accuracy | 99.94%    | 99.91%     |
| Recall   | ~0%       | **86%** ✅  |
| F1-Score | ~0%       | **0.79** ✅ |
